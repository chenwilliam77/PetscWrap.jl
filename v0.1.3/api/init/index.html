PetscWrap.PetscInitialize — Function
PetscInitialize(cmd_line_args::Bool = true)

Initialize PETSc.

If cmd_line_args == true, then command line arguments passed to Julia are used as arguments for PETSc (leading to a call to PetscInitializeNoPointers).

Otherwise, if cmd_line_args == false, initialize PETSc without arguments (leading to a call to PetscInitializeNoArguments).
